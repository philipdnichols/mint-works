# Mint Works!

Mint Works is a light worker placement game for 1 to 4 players where you develop a Neighborhood by gaining and building Plans to score Stars. This project is a browser-based implementation of the tabletop game.

Live site: https://<username>.github.io/mint-works/

## How To Play (Summary)

- Set up the Core and Deed Locations, shuffle the Plan Deck, and reveal the Plan Supply.
- In the Development Phase, take turns placing Mints on Locations to gain Plans, build Buildings, and take bonuses.
- In the Upkeep Phase, check for end game, refill the Plan Supply, resolve Upkeep effects, clear Locations, and gain income.
- When the game ends, total Stars from Buildings. The highest total wins.
